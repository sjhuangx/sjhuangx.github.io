Giscus评论

Giscus是一个简单的博客评论系统，基于Github自身的Discussions，本身的设置较为简单，需要提供一个仓库满足下面三个条件即可
* 仓库是公开可访问的
* giscus app已安装否则访客将无法评论和回应。
* Discussions功能已在你的仓库中启用。

阅读全文

如果不做任何配置，默认会在首页将全文展示出来，如果文章很长的话会占据首页很长的篇幅，用户需要手动滚动很长才能看到下一篇文章，所以对于较长的文章可以只展示部分的内容，然后添加一个【阅读全文】的按钮，如果用户对文章有兴趣的话就点击该按钮阅读全文，对于Next主题来说，有个非常简单的方法添加【阅读全文】按钮，就只在文章内容中添加<!-- more -->标签即可，这也是hexo推荐的方法。
